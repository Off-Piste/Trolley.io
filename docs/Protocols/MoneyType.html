MoneyType

public protocol MoneyType

The Protocol that holds the properties that can be essential for creating a value that works like money

  • amount

    The Float value for the decimal number found inside: 

    <DecimalType>.money.value

    The value is converted to the currency that is found inside the same tuple as above

    Declaration

    Swift

    var amount: Float
  • floatValue

    The non-converted float value, this would be used by developers that are wanting to do custom conversions of the money or convert the money prior

    Declaration

    Swift

    var floatValue: Float
  • integerValue

    The non-converted int value, this would be used by developers that are wanting to do custom conversions of the money or convert the money prior.

    Note:

    Rememeber that the values will lose there decimal place:

    var money: Money = 50.55
print(money.integerValue) 
// prints ("50\n")

    To get int value but with decimal use .stripe

    Declaration

    Swift

    var integerValue: Int
  • stripe

    The non-converted int value that returns the value in cents rather than dollars. Helpful when working with Stripe as their API takes the value as cents

    var money: Money = 50.55
print(money.stripe)
// prints ("5050\n")

    To get int value but without the deicmal use .integerValue

    Declaration

    Swift

    var stripe: Int
  • string

    The formatted string value for the money, this takes the converted value and creates a formatted value based on the currency. This is so the user doesn’t have to.

    var money: Money = 50.55
print(money.string, money.currency.code)
// prints ("£47.90 GBP\n")

    Declaration

    Swift

    var string: String
  • currency

    The gateway to the Currency enum, this allows users to get the current currency code and conversion rate

    Declaration

    Swift

    var currency: Currency
  • decimalValue

    The converted NSDecimalNumber value of the currency.

    Declaration
